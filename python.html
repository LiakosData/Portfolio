Python Stock Analysis

This project is still under development. It is a Streamlit application that calculates important stock metrics, including Sharpe Ratio, Yearly Returns, stock price, RSI, and Moving Averages.
Users can select any stock from the list and see the metrics and graphs change based on their selection.
Several improvements are planned, including forecasting with ML models, portfolio optimization, and support for different timeframes.
The list of stocks will be significantly expanded in the final version.
