Project: Stock Market Analysis using Python & Power BI

This project analyzes six major stocks (AAPL, AMZN, NVDA, TSLA, GOOG, SPY) over the last 5 years using Python and Power BI. It includes trend analysis, volatility measurement, and correlation insights.

Technologies: Python (Pandas, Matplotlib, Seaborn, yfinance, talib), Power BI (DAX, interactive visuals), Jupyter Notebook.

Key Features:

  • Data Cleaning & Preprocessing
  • Calculation of Key Metrics (MA, RSI, Bollinger Bands)
  • Correlation Analysis
  • Interactive Power BI Reports
